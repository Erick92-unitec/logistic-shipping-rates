
## Paso 4: Crear CONTRIBUTING.md

GitHub permite colocar las pautas de contribución en la raíz del repositorio con el nombre `CONTRIBUTING.md`. :contentReference[oaicite:2]{index=2}

```markdown
# Contributing Guidelines

Thank you for your interest in contributing to this project.

## How to Contribute

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Submit a pull request.

## Code Standards

- Use clear variable names.
- Keep functions simple and readable.
- Add comments when the logic is not obvious.
- Follow Python best practices.

## Reporting Issues

When reporting an issue, please include:

- A clear title.
- Steps to reproduce the problem.
- Expected result.
- Actual result.
- Screenshots or logs, if available.
