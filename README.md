# Logistics Shipping Rates

This project provides an initial Python script to calculate logistics shipping rates based on package weight and shipping distance.

## Features

- Calculates shipping rates using weight and distance.
- Provides a simple command-line interface.
- Designed as an initial open-source logistics calculator.

## Usage

```bash
python shipping_rates.py
